# Shopfy - E-commerce Platform

Multi-user e-commerce platform with customer, store owner, and admin roles.

## Features
- User authentication and authorization
- Store management
- Product catalog
- Shopping cart and checkout
- Order management
- Admin dashboard
- Profile update requests
- Support ticket system
- Login/logout statistics

## Tech Stack
- **Frontend**: React, Vite
- **Backend**: Spring Boot, Java
- **Database**: MySQL
- **Authentication**: Session-based

## Local Development

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
